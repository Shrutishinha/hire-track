# 🚀 HireTrack — Smart Job Application Tracker

HireTrack is a full-stack MERN application designed as a **personal CRM for job seekers**. It helps users manage job applications, track progress, analyze skill gaps, and compare offers — all in one place.

---

## 📌 Features

- 📊 **Dashboard Overview**
  - Track total applications, pipeline status, and active offers

- 📌 **Kanban Pipeline**
  - Drag-and-drop job tracking across stages:
    - Applied → Screen → Interview → Offer → Accepted → Rejected

- 🔍 **Multi-Source Job Feed**
  - Aggregates and filters job listings based on user skills

- 🤖 **AI Skill Gap Analyzer**
  - Suggests missing skills required for target roles

- 📄 **Resume Parser**
  - Upload PDF → Extract skills and auto-fill profile

- 🔔 **Smart Notifications**
  - Alerts for high-match job opportunities

- ⚖️ **Offer Comparison**
  - Compare multiple offers side-by-side

- 🔐 **JWT Authentication**
  - Secure login and session management
